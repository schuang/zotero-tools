# Zotero tools:

## ref-to-zotero

This is a Chrome browser extension to add references to Zotero online library.

While there is already a [Zotero connector](https://www.zotero.org/download/connectors), but I need an easy way to preview the meta data before adding it to Zotero library.

## Zotero manager

A collection of scripts to perform certain operations on the Zotero library
